Orbeon Forms
============

## Install Tomcat

`sudo apt-get install tomcat6 tomcat6-admin`

`sudo vi /etc/default/tomcat6`
`TOMCAT6_SECURITY=no`

## Install Orbeon

`cd /var/lib/tomcat/


## Install Orbeon Resource

1. `cd /srv/gather`
2. `tar xvzf orbeon-resource.tgz`
